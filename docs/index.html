Quick Setup — connect your machines in 5 minutes ▾
  1. 1
    Install pip install gpuwatch
    or download single file: curl -O https://raw.githubusercontent.com/reacher-z/gpu-monitor/main/gpu_monitor.py
  2. 2
    Create a GitHub token Go to github.com/settings/tokens → New token (classic) → tick public_repo scope.
  3. 3
    Set environment variables export GITHUB_PAGES_REPO=your-username/gpu-monitor
    export GITHUB_PAGES_TOKEN=ghp_...
  4. 4
    Run on each machine gpu-monitor
    Stats are pushed every 30 seconds and appear here automatically. You'll see a card per machine with GPU utilization, memory, temperature, and running processes in real time.
  5. 5
    Add Slack / Discord alerts (optional) export SLACK_WEBHOOK_URL=https://hooks.slack.com/...
    Run gpu-monitor --channels to see all 20 options.
No dashboard data is stored on any server. Stats are committed as JSON files directly to your GitHub repository — the same one hosting this page.
