Quick Setup — up and running in 3 steps ▾
  1. 1
    Install
    pip install gpuwatch
  2. 2
    Set a notification channel — pick any one:
    export SLACK_WEBHOOK_URL=https://hooks.slack.com/services/... # or Discord export DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/... # or ntfy (zero signup — subscribe on your phone) export NTFY_URL=https://ntfy.sh/my-gpu-alerts-abc123
    Run gpu-monitor --channels to see all 20 options.
  3. 3
    Run
    gpu-monitor
    You'll get an alert immediately on startup, then again on crash, OOM, overheat, or idle.
    Optional — live web dashboard: fork this repo, enable GitHub Pages, then set GITHUB_PAGES_REPO=your-username/gpu-monitor and GITHUB_PAGES_TOKEN=ghp_.... Stats push automatically every 60 seconds.
